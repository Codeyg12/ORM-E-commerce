  ### Installation

To install follow these steps

1.  Clone the repo using either HTTPS

```sh
https://github.com/Codeyg12/ORM-E-commerce.git
```

or SSH key

    git@github.com:Codeyg12/ORM-E-commerce.git

2.  install NPM packages

```sh
npm i
```

3.  Run the seeds file

```sh
npm run seed
```

4.  Start the server

```sh
npm start
```
